# EFREI Web Project - Plateforme Académique

Ce projet est une plateforme web interactive développée dans le cadre du cursus à l'EFREI. Elle regroupe des informations sur les formations, les enseignants, la recherche et un planning dynamique des cours et permanences.

## 🚀 Démo en ligne
Le site est hébergé sur GitHub Pages :
[Consulter le site ici](https://benoittailhades.github.io/Web_project/)

## ✨ Fonctionnalités clés
- **Navigation Animée** : Barre de navigation avec un curseur fluide qui suit l'utilisateur.
- **Planning Dynamique** : Système de calendrier généré en JavaScript avec mise en évidence automatique du jour actuel.
- **Design Responsive** : Adapté aux mobiles et tablettes grâce aux Media Queries CSS.
- **Profils Enseignants** : Pages dédiées avec détails des recherches et horaires de permanence (Office Hours).

## 🛠️ Technologies utilisées
- **HTML5** : Structure sémantique.
- **CSS3** : Flexbox, Grid, Animations et Variables (Custom Properties).
- **JavaScript (ES6)** : Manipulation du DOM, gestion du calendrier et animations de navigation.

## 📱 Tester le mode Responsive
Pour tester le rendu mobile :
1. Connectez votre téléphone au même réseau Wi-Fi que votre ordinateur.
2. Trouvez l'IP de votre machine (`ipconfig` sur Windows).
3. Accédez à `http://[VOTRE_IP]:5500` (si utilisation de Live Server).

## 📂 Structure du projet
- `/HTML` : Pages de contenu (home, planning, contact, etc.).
- `/css` : Feuilles de style (design global et responsive).
- `/js` : Scripts logic (calendrier, animations).
- `/img` : Ressources graphiques et logos.

## ⚖️ Conformité
- Respect des standards W3C.
