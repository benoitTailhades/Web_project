Principal Component Analysis and Diagonalization

Signal Processing Laboratory

The Curse of Dimensionality

Handling data matrices containing a high number of variables poses strict computational challenges. Principal Component Analysis (PCA) is a geometric basis transformation method. The goal is to project the data onto a lower-dimensional vector subspace while retaining maximum variance (information).

Covariance Matrix and the Eigenvalue Problem

Considering a centered data matrix $X$, the empirical covariance matrix is given by $\Sigma = \frac{1}{n-1} X^T X$. Since $\Sigma$ is a real symmetric matrix, the spectral theorem guarantees that it is diagonalizable in an orthonormal basis. The eigenvectors of this matrix form the principal axes, and the associated eigenvalues quantify the variance captured by each axis.

"Retaining the $k$ eigenvectors associated with the largest eigenvalues compresses information with minimal loss."

Applications in Preprocessing

PCA serves primarily as a preprocessing phase. By reducing the collinearity of explanatory variables, it conditions the system such that subsequent classification algorithms converge faster, thereby avoiding overfitting while saving the infrastructure's memory resources.
