Splitting Criteria via Shannon Entropy

Algorithmic Research Group

Information Theory and Trees

Building a decision tree involves iteratively choosing the variable that best separates a dataset according to their classes. The quality of this separation is measured using Shannon entropy, a concept derived from probability and information theory that quantifies the disorder or uncertainty within a set.

Mathematical Information Gain

For a dataset $S$ containing observations distributed across $c$ classes, entropy is defined as $H(S) = - \sum_{i=1}^{c} p_i \log_2(p_i)$, where $p_i$ is the proportion of elements in class $i$. The algorithm's objective is to find the split that maximizes information gain, which is the difference between the entropy of the parent node and the weighted sum of the children nodes' entropies.

"Minimizing entropy at each node creates increasingly homogeneous space partitions from a probabilistic standpoint."

AL
Antoine Lemaire
Postdoctoral Researcher

Complexity Control

The unconstrained development of such a tree inevitably leads to each leaf containing a single element. Although the entropy is then zero ($H(S) = 0$), this model proves useless for generalization. Pruning techniques based on complexity penalty criteria are added to halt the growth of the tree structure.
